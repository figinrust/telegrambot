# telegrambot

Телеграм бот "Таскер" напоминалка для задач

Функции:

1. Добавление задач
2. Удаление задач которые просроченны по времени
3. Просмотр задач в списке задач
4. Функция сброса
5. Удаление задач(включая выключение таймера thread)
